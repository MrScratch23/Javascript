5c - Interacción con CSS

Contenido básico:

Gestión simplificada

Se puede manejar hasta cierto punto el CSS desde JS. Partimos de esto: 

let btn = document.querySelector("#btnAccion");

Podemos añadirle estilos en línea así: btn.style="outline: 2px solid red;"


Se puede acceder y cambiar las clases de un elemento: 

btn.className = "clase1 clase2" // como cadena, recomendado para cambiar la lista completa 
                                // de clases de una vez

Pero no está permitido esto: 

btn.classList[1] = "clase1"; // sólo la lectura está permitida

btn.classList = ["clase1", "clase2"] // no funcionará como esperamos, el navegador intentará
                                     // convertir  el array a cadena de caracteres y asignará
                                     // una única clase "clase1,clase2"

También podemos acceder a sus clases y modificarlas con btn.classList de una manera más granular y segura:

  • btn.classList.add("rojo", "borde1"): agrega las clases rojo y borde1 al botón.
  • btn.classList.remove("rojo"): elimina la clase rojo
  • btn.classList.toggle("rojo"): alterna la presencia de la clase (si la clase ya está presente, la elimina, si no está presente, la agrega)
  • btn.classList.contains("rojo"): devuelve true o false si el elemento tiene asignada la clase

También hay otros más circunstanciales:

  • btn.classList.replace("rojo", "verde"): Reemplaza la rojo clase por la verde
  • btn.classList.item(0) o btn.classList[0]: devuelve la clase CSS que esté en la posición indicada (en el ejemplo es la primera), puede ser usada para iterar en un bucle por lass clases. No se pueden modificar las clases de esta manera.
  • btn.classList.length: el número de clases CSS asignadas al elemento.

Usando la propiedad style

Se puede usar de dos maneras, modificando toda la cadena style o sólo una propiedad concreta.


Nota: si la propiedad tiene guiones se convierte en camelCase, por ejemplo text-align → textAlign.


Ejemplo: 

elem.style = "background-color: yellow;";
elem.style.backgroundColor = "yellow";
elem.style.paddingLeft = "10px";   //hay que indicar la unidad

// Eliminación:
elem.style.backgroundColor = "";

Nota: Por las reglas de precedencia de CSS, estas propiedades tienen más prioridad que las definidas en un fichero CSS.


Nota: si queremos recuperar la cadena de texto de los estilos inline, podemos usar elem.style.cssText, porque elem.style nos devuelve un objeto.


Nota: una excepción a la regla camelCase es la propiedad CSS float que se convierte a cssFloat (float es una palabra reservada de Javascript).
Ejemplo elem.style.cssFloat = "right";




Propiedades de estilo con valores computados


Referencia: https://www.w3.org/TR/1998/REC-CSS2-19980512/cascade.html#computed-value


Hay propiedades que permiten valores absolutos como 5px o blue, pero también valores cuyo valor hay que calcular como auto, em, 10% cuyo valor depende de las características del dispositivo y tienen que ser calculados.


La propiedad style no nos permite conocer los estilos computados, pero podemos obtenerlos usando el método getComputedStyles(elemento).

  • El objeto que devuelve es de solo lectura.
  • Solo se pueden recuperar propiedades no compuestas.

Ejemplo: 

let oEstilo = getComputedStyle( elemento );
console.log( oEstilo.width );       // "317.583px"
console.log( oEstilo.marginTop );   // "10px"
console.log( oEstilo.margin );      // ""



Objetos que describen ficheros y bloques de estilo


En document.styleSheets podemos obtener un array de los ficheros CSS y los bloques de estilo incluidos en nuestra página.

  • document.styleSheets[index] – Es una hoja de estilos concreta
  • document.styleSheets[0].href – Si es una hoja externa, esta es la URL de la misma (puede ser útil para diferenciar varias hojas de estilo)
  • document.styleSheets[index].cssRules – Lista de reglas CSS (no es un array, aunque tenga la propiedad length y se pueda usar el operador []). Hay que tener en cuenta que no podremos cambiar de esta manera las reglas que provengan de dominios externos (un CDN, por ejemplo).

Podemos desactivar una hoja de estilos así: 

document.styleSheets[0].disabled=true

El array de reglas contiene objetos cssRule con las siguientes propiedades:

  • cssText – texto completo de la regla (no se puede modifica aquí)
  • selectorText – texto del selector (sí puede ser modificado)
  • objeto style – que nos permite conocer y modificar los estilos aplicados.
    • Ejemplo: document.styleSheets[0].cssRules[5].style.backgroundColor="yellow"
    • También se puede usar el método setProperty()
      • Ejemplo de lectura: getComputedStyle(elemento).getPropertyValue('--nombre-variable)
      • Ejemplo de escritura: elemento.style.setProperty('--nombre-variable', 'nuevo valor')

Podemos utilizar el método insertRule para crear dinámicamente reglas nuevas en una hoja de estilos y deleteRule para eliminarlas:
Ejemplo: 

// El 0 indica el índice, en este caso se inserta al principio 
// (si no lo indicamos se inserta como primera regla igualmente)
document.styleSheets[index].insertRule("#blanco { color: white }", 0); 

document.styleSheets[index].deleteRule( 0 ) // borra la primera regla

Importante: si la hoja de estilos usa @import, por ejemplo para añadir tipografías externas, no se podrá añadir una nueva regla antes de algún de estos imports (los imports deben aparecer primero).




Cambiar la hoja de estilos por otra

En el HTML indicamos un id al elemento link: 

...
<head>
  <!-- Light theme stylesheet -->
  <link href="light-theme.css" rel="stylesheet" id="theme-link">
</head>
...

Al pulsar el botón se cambia la hoja de estilo que se carga: 

document.querySelector("#btnTema").addEventListener("click", () => {
  // seleccionamos la etiqueta link
  const theme = document.querySelector("#theme-link");

 // cambiamos el fichero css que se usa
 if (theme.getAttribute("href") == "light-theme.css") {
  theme.href = "dark-theme.css";
 } else {
  theme.href = "light-theme.css";
 }
});



Cambiar al modo oscuro usando variables

Usando variables CSS en el :root 

:root{
  --border: blue;
  --accent-bg: aliceblue;
}

:root.dark {
  --border: navy;
  --accent-bg: rgb(199, 208, 215);
}

Al pulsa el botón se añade la clase dark: 

document.querySelector("#btnTema").addEventListener("click", () => {
  document.querySelector(":root").classList.toggle("dark");
});






Ejercicios

Ejercicio 1: crea una web que use una hoja de estilos que asigne a los botones y al body estas reglas CSS: 

body{
  background-color: #eee;
  color: #444;
}

button{
  padding: .5rem;
  margin-bottom: .5rem;
  border-radius: 5px;
  max-width: 100%;
  color: white;
  background-color: #0d47a1;
  border: 1px solid black;
}

Haz que cuando se pulse un botón con la etiqueta Color, se cambie el color de fondo de los botones a #47adad;


Por otro lado, crea un segundo botón con la etiqueta Modo oscuro/claro que cuando se pulse cambie la hoja de estilos actual a otra hoja de estilo diferente con las reglas: 

body{
  background-color: #456;
  color: #def;
}
button{
  padding: .5rem;
  margin-bottom: .5rem;
  border-radius: 5px;
  max-width: 100%;
  color: white;
  background-color: #0d47a1;
  border: 1px solid black;
}

Si se pulsa el botón de cambiar el modo, se volverá a cargar la hoja de estilos inicial.



Ejercicio 2: Crea una web que use tailwind a partir de un CDN. Según la documentación inicial (https://tailwindcss.com/docs/installation/play-cdn), esto se consigue añadiendo al <head> la siguiente línea. 

<script src="https://cdn.tailwindcss.com"></script>

En tu web habrá una etiqueta <p id="parrafo">Hola caracola</p> y un formulario con un textbox y dos botones: Añadir y Eliminar.
Cada vez que se pulse Añadir, se añadirá al párrafo la clase de tailwind que el usuario haya indicado en el textbox.
Cada vez que se pulse Eliminar, se eliminará del párrafo la clase de tailwind que el usuario haya indicado en el textbox.


Ejemplos de clases para probar: 

p-2
m-5
bg-indigo-200
w-1/2
border-solid border-2 border-indigo-500 
drop-shadow-lg


Ejercicio 3: Crea una web que contenga un cuadro de texto y un botón con la etiqueta Añadir que añada el contenido del cuadro de texto a una lista.
También habrá un botón con la etiqueta Colorear. Al pulsar este botón se colorearán los ítems de la lista con estilos en línea (con la etiqueta style) de la siguiente manera:

  • Los que tengan un número de caracteres impar tendrán un color de fondo rojo y letras en blanco.
  • Los que tengan un número de caracteres par tendrán un color de fondo goldenrod.

Resuelve el ejercicio también añadiendo reglas CSS dinámicamente a la hoja de estilos de la página: 

li:nth-child(odd) { background-color:goldenrod; color:white;}
li:nth-child(even){ background-color:red; color:white;}
